# web_development_task_2
## To-do List code ##
# Overview
A simple and dynamic To-Do List application built with HTML, CSS, and JavaScript. Users can add, remove, and mark tasks as complete, providing an intuitive way to manage daily tasks.

# Tools Used
Visual Studio Code (Code Editor)
Live Server Extension for Real-time preview.

# Features
Add Tasks – Users can input tasks and add them to the list.
Remove Tasks – Users can remove tasks they no longer need.
Mark Tasks Complete – Clicking a task will toggle its completion status.
Responsive Design – Works seamlessly across different screen sizes.

# Deliverables
index.html – Structure of the To-Do list.
style.css – Styling for a clean and modern UI.
script.js – JavaScript logic for task management.

# Usage
1. Clone the repository or download the files.
2. Open the project folder in Visual Studio Code.
3. Use live-server extension to  run the application.
